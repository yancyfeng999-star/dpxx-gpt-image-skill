## 一句话版

你只要告诉我四件事：

```text
我要画什么？
用在哪里？
想要横图、竖图还是方图？
有没有参考图或必须出现的文字？
```

例如：

```text
帮我做一张小红书封面，主题是办公室咖啡，竖图，风格高级干净，要有文字“早八续命咖啡”。
```

## 开始前：让 agent 安装技能

第一次使用前，需要先让 agent 安装或更新 **dpxx-image-skill v1.1.1** 技能。你可以直接把下面这段话发给 agent：

```text
请帮我安装或更新 dpxx-image-skill v1.1.1 技能。
GitHub 开源仓库是：https://github.com/yancyfeng999-star/gpt-image-2.git
安装或更新完成后，请告诉我已经可以用它生成图片。
```

如果 agent 需要手动执行命令，可以让它直接运行下面这一段。已安装时会更新，未安装时会安装：

```bash
mkdir -p ~/.claude/skills
if [ -d ~/.claude/skills/dpxx-image-skill/.git ]; then
  git -C ~/.claude/skills/dpxx-image-skill pull --ff-only
else
  git clone https://github.com/yancyfeng999-star/gpt-image-2.git ~/.claude/skills/dpxx-image-skill
fi
```

安装或更新完成后，再继续准备 API Key 和出图需求。

## 开始前：获取 API Key

使用 dpxx-image-skill 出图前，需要先准备 RootFlowAI API Key。

API Key 只分 GPT 和 Gemini。GPT-Image-2 用 GPT API Key；Gemini 3.1 Flash / Gemini 3 Pro 用 Gemini API Key。API Key 请找 **YancyFeng 工程师** 获取，拿到后，把对应模型的 API Key 发给负责出图的 agent 或工程师即可。

API 地址不用你区分，GPT 和 Gemini 都走同一个 RootFlowAI 地址；系统只通过不同 API Key 做统计和分流。

## 第一步：先选模型

使用 skill 开始先选模型。这里只有 3 个顶层选项，不需要在这一步选 1K / 2K / 4K。不会选时，直接选 `GPT-Image-2`；如果你明确想用 Gemini，选 `Gemini 3.1 Flash`。

| 选择 | 适合什么 | 建议 |
|------|----------|------|
| GPT-Image-2 | 通用出图、商品图、海报、现有默认风格 | 默认推荐 |
| Gemini 3 Pro | 复杂海报、图片里有文字、专业资产 | 复杂图优先 |
| Gemini 3.1 Flash | 想用 Gemini / Nano Banana，重视提示词跟随 | Gemini 默认推荐 |

## 第二步：先选分辨率

分辨率决定清晰度和可选比例。不会选时，直接选 `2K`。

| 选择 | 适合什么 | 建议 |
|------|----------|------|
| 1K | 快速试想法、头像、草图、社媒小图 | 想先看看方向时用 |
| 2K | 海报、封面、商品图、常规成品 | 默认推荐 |
| 4K | 超清大图、宽幅海报、大屏、精修最终稿 | GPT-Image-2 只支持部分宽幅比例 |

简单理解：

```text
先试想法：1K
大多数正式图：2K
超清宽幅最终图：4K
```

## 第三步：选图片比例

比例决定图片形状。不会选时，看你准备用在哪里。

| 用途 | 推荐比例 |
|------|----------|
| 头像、Logo、图标 | 1:1 |
| 小红书封面、手机海报、竖版宣传图 | 3:4、4:5、9:16 |
| 商品主图、电商详情页局部 | 1:1、4:5、3:4 |
| 横版海报、公众号头图、B 站封面 | 16:9、3:2 |
| 超宽横幅、网页 Banner、大屏背景 | 21:9、2:1 |

GPT-Image-2 4K 只能选这些宽幅比例：

```text
16:9, 9:16, 2:1, 1:2, 21:9, 9:21
```

如果你想用 GPT-Image-2 做 `1:1` 方图，就不要选 4K，选 1K 或 2K。Gemini 4K 可以优先尝试常见比例，例如 `1:1`、`16:9`、`9:16`、`21:9`。

## 第四步：描述你想画什么

一个好需求不需要很长，但最好包含这些信息：

```text
主体：画什么
用途：用在哪里
风格：看起来像什么
文字：图片里必须出现哪些字
限制：不要什么
```

可直接套用这个格式：

```text
我要做一张【用途】，主题是【主体】。
画面风格要【风格】，整体感觉【情绪】。
比例用【比例】，分辨率用【1K/2K/4K】。
图片里需要出现文字：【文字】。
不要出现：【不想要的内容】。
```

例子：

```text
我要做一张小红书封面，主题是秋天第一杯热拿铁。
画面风格要温暖、干净、有高级感，整体感觉像生活方式杂志。
比例用 4:5，分辨率用 2K。
图片里需要出现文字：秋天第一杯热拿铁。
不要出现人物、杂乱背景、水印。
```

## 第五步：选择风格

如果不知道要什么风格，可以从下面挑一个词：

```text
写实摄影
电影感
日系清新
高级商业广告
极简设计
可爱插画
国潮
赛博朋克
复古海报
电商详情页
品牌视觉板
UI 界面
```

也可以说：

```text
你帮我推荐 3 个风格方向，我来选。
```

## 关于图片质量

默认就是高质量，不用特别说。

只有你明确想要低质量预览时，才需要说：

```text
先低质量预览一下
```

默认高质量更省心；只有明确想快速低质量预览时，再说“先低质量预览一下”。

## 生成前确认

agent 执行前必须先把三件事发给你确认：

```text
参数摘要：模型、分辨率、比例、模板
中文提示词：你能看懂的完整描述
English prompt：实际传给模型的英文提示词
```

你确认回复“执行”之后，agent 才会开始调用脚本。

## 有参考图怎么说

如果你有参考图，可以直接发图，并说明“保留什么、改变什么”。

好例子：

```text
参考这张照片的人物姿势和衣服，改成电影海报风格。
保留人物脸型和发型，背景换成雨夜街道。
比例 2:3，2K。
```

再比如商品图：

```text
用这张产品图做电商主图。
保留产品外形和包装文字，背景换成白色高级棚拍，增加柔和阴影。
比例 1:1，2K。
```

## 想改图怎么说

看到第一张图后，不满意很正常。改图时不要只说“不好看”，最好说清楚要改哪里。

可以这样说：

```text
背景太乱，换成纯白背景。
人物表情太严肃，改得更自然一点。
文字太小，放大并放在上方。
颜色太暗，整体更明亮。
Logo 方向对了，但图标再圆润一点。
```

如果只想局部修改，说明保留区域：

```text
只改背景，人物和衣服保持不变。
只改文字排版，画面主体不要动。
只把杯子颜色改成红色，其他不变。
```

## 常见需求模板

### Logo

```text
帮我做一个 Logo，品牌名是【品牌名】。
行业是【行业】，关键词是【3 个关键词】。
风格要【极简 / 科技 / 可爱 / 高级 / 国潮】。
比例 1:1，分辨率 2K。
```

### 小红书封面

```text
帮我做一张小红书封面。
主题是【主题】，目标人群是【人群】。
画面要【风格】，需要大标题【标题】。
比例 4:5，分辨率 2K。
```

### 商品主图

```text
帮我做一张商品主图。
产品是【产品】，卖点是【卖点】。
风格要高级、干净、商业摄影感。
比例 1:1，分辨率 2K。
```

### 海报

```text
帮我做一张海报。
主题是【活动 / 电影 / 产品 / 课程】。
主视觉是【画面主体】，需要文字【标题和副标题】。
风格是【电影感 / 复古 / 科技 / 极简】。
比例 2:3，分辨率 2K。
```

### 电商详情页

```text
帮我做一张电商详情页视觉图。
产品是【产品】，需要突出【3 个卖点】。
画面包含产品、卖点标签、使用场景。
比例 3:4 或 4:5，分辨率 2K。
```

## 常见坑

1. 不要只说“帮我画得高级点”。最好说高级在哪里：配色、光影、排版、材质、留白。
2. 图片里有文字时，一定把文字逐字写出来。
3. 想要方图就选 1:1，不要选 4K。
4. 想保留参考图里的某些东西，一定写“保留 XXX”。
5. 想避免某些东西，一定写“不要 XXX”。

## 最省事的提问方式

直接复制下面这段填空：

```text
我要做：【图片用途】
主体是：【画什么】
风格是：【想要的感觉】
比例：【不知道就写“你推荐”】
分辨率：【不知道就写“2K”】
模型：【不知道就写“GPT-Image-2”；想用 Gemini 就写“Gemini 3.1 Flash”】
必须出现的文字：【没有就写“无”】
参考图要求：【没有就写“无”】
不要出现：【没有就写“无”】
```

示例：

```text
我要做：小红书封面
主体是：一杯热拿铁和一本打开的书
风格是：秋天、温暖、干净、生活方式杂志感
比例：4:5
分辨率：2K
必须出现的文字：周末慢下来
参考图要求：无
不要出现：人物、水印、杂乱桌面
```
