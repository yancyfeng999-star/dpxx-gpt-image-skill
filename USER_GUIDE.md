# GPT Image User Guide

Use this skill when you want DPXX GPT image generation through RootFlowAI.

Tell the agent:

- What to draw or edit.
- Resolution: `1K`, `2K`, or `4K`.
- Ratio: for example `1:1`, `4:5`, `16:9`, or `9:16`.
- Whether reference images are needed.

Default choices:

- Model: `gpt-image-2-hd-count`.
- Quality: `high`.
- Ratio: choose based on the final use case.

For 4K, choose only `16:9`, `9:16`, `2:1`, `1:2`, `21:9`, or `9:21`.
